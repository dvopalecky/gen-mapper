Advice is needed from fellow disciple-makers!
Below are a few new versions of the app. Please try them and give feedback.
Email: gospel@tuta.com.

NOTE: Older disciple maps from the OG version won't import into the new
disciple mapper versions beause the new versions require more headers in
the csv(excel) file that stores the disciples info. You will have to update
the headers of the old CSV file from the OG version to match the CSV file
structure for the new disciple mapper versions. If you need help with this
there should be a tutorial coming out soon. Email for help too.
